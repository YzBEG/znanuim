# 🎓 ZNANIUM - Платформа для поиска репетиторов

Современная веб-платформа для поиска репетиторов с системой бронирования уроков, отзывами и real-time чатом.

---

## 🚀 Быстрый старт

### Запуск проекта

```bash
# 1. Активировать виртуальное окружение
.venv\Scripts\activate

# 2. Запустить сервер (с WebSocket поддержкой)
daphne -b 127.0.0.1 -p 8001 config.asgi:application

# 3. Открыть в браузере
http://127.0.0.1:8001/
```

### Основные URL

| Страница | URL |
|----------|-----|
| Главная | http://127.0.0.1:8001/ |
| Каталог репетиторов | http://127.0.0.1:8001/catalog/ |

| Регистрация | http://127.0.0.1:8001/accounts/register/ |

| Панель модерации | http://127.0.0.1:8001/users/admin/moderation/ |
| REST API | http://127.0.0.1:8001/api/ |

## 🎯 Основной функционал

### Для студентов
- ✅ Регистрация и авторизация
- ✅ Поиск репетиторов с фильтрами (предмет, формат, город, цена)
- ✅ Просмотр профилей и отзывов
- ✅ Бронирование уроков
- ✅ Real-time чат с репетитором
- ✅ Оставление отзывов после уроков
- ✅ Личный кабинет с историей

- ✅ Регистрация и создание анкеты
- ✅ Управление расписанием (слоты)
- ✅ Подтверждение/отклонение заявок
- ✅ Проведение уроков
- ✅ Загрузка материалов к урокам
- ✅ Ответы на отзывы (с возможностью редактирования)
- ✅ Просмотр статистики и отзывов
- ✅ Список учеников

### Для администраторов
- ✅ Модерация анкет репетиторов
- ✅ Управление заявками с сайта
- ✅ Просмотр всех данных
